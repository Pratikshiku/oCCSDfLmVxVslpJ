## ERP Warehouse Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/4e60774245dd4558b05180fc5edb6321/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (any of the listed software can be used)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Functional Description
The system is divided into (roles): regular users, warehouse employees, administrators.

**Regular User Functions:**
1. Login
2. Register
3. Personal Center
4. Change Password
5. View Announcements
6. View Materials
7. Material Applications

**Warehouse Employee Functions:**
1. Login
2. Register
3. Personal Center
4. Change Password
5. View Announcements
6. View Materials
7. Material Applications

**Administrator Functions:**
1. Login
2. Personal Center
3. Change Password
4. Administrator Management
5. Basic Data Management (warehouse management, announcement type management, material type management, material application type management)
6. Announcement Management
7. Material Management
8. Material Application Management
9. User Management
10. Warehouse Administrator Management
